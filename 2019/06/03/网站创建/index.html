1、hexo

1.1 hexo 安装

  • hexo是一个静态博客生成工具，基于nodejs就能创建只有前端的网站，类似hexo的还有Wordpress、Typecho。
  • 搭建博客原因是之前一直都想写一个自己的博客，但是自己是做后端的，搭建网站需要前端后端一起搞，涉及数据库、服务器等，这种网站可以虽然可以交互，但是麻烦。用hexo只需要搭建一下界面，然后用markdown写文章直接发布到github上就行。
